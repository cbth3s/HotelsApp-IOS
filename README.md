## üì≤ Install:
### Backend(Optional):
1. Clone the repository with backend:
```bash
git clone https://github.com/cbth3s/HotelsApp-Backend
```
2. Activate the virtual environment:
```bash
cd HotelsApp-Backend
python3 -m venv .venv
source .venv/bin/activate
```
3. Install dependencies:
```bash
pip install -r requirements.txt
```
4. Run:
```bash
python main.py 
```
### iOS project:
1. Clone the repository:
```bash
git clone https://github.com/cbth3s/HotelsApp-IOS
```
2. Open in Xcode:
```bash
cd HotelsApp-IOS
open Hotels.xcodeproj
```
3. Run: `Command + R`
